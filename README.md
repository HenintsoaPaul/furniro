# Furniro - Nuxt Test App

This is a test app implementing a figma template. 

## Run commands

### In a container (recommended)

Require docker to be installed.

```bash
    docker compose up
```

### Locally

Require node and npm to be installed.

```bash
    npm install
    npm run dev
```
